Events

clickoutside event

Triggered on an event when the mouse is clicked outside that element.

Usage

jQuery('selector').bind( 'clickoutside', function(event) {
  var clicked_elem = $(event.target);
  ...
});

Additional Notes

  • Because the ‘clickoutside’ event utilizes the ‘click’ event internally, if the ‘click’ event has its event propagation stopped on a clicked element for some reason, it will prevent ‘clickoutside’ from triggering on any related elements.
